# SUPER-MARIO11 Portfolio

Fast, responsive GitHub portfolio website built with HTML, CSS, and Vanilla JS.

## Links
- Portfolio: https://super-mario11.github.io
- Profile: https://github.com/Super-mario11
- Profile README repo: https://github.com/Super-mario11/Super-mario11
- Profile README git URL: `https://github.com/Super-mario11/Super-mario11.git`
- Profile README file: https://github.com/Super-mario11/Super-mario11/blob/main/README.md

## Features
- Live GitHub profile data
- Top repositories (limited to 3)
- Recently added repositories with stats and last activity
- GitHub stats + contribution graph
- Recent activity timeline
- OneFetch-style snapshot
- Dark/light mode and lazy-loaded stat media

## Stack
- HTML
- CSS
- Vanilla JavaScript
- GitHub Pages

## Deploy (GitHub Pages)
1. Push this project to `super-mario11.github.io`.
2. In GitHub, go to `Settings -> Pages`.
3. Set source to `Deploy from a branch`.
4. Select `main` and `/ (root)`.

## Notes
- GitHub API unauthenticated rate limits apply.
- External stat image services may occasionally return temporary `5xx` responses.
